# Simple Display

Player companion display for HVS tabletop setup. Requires Foundry VTT v14+.

## Setup

1. Install the module via the manifest URL or manually
2. In **Game Settings → Module Settings → Simple Display**, paste the Actor ID for each display
3. Click the tablet icon in the scene toolbar to open a display

## Development

- `scripts/` — ES modules, ApplicationV2-based
- `styles/` — CSS with custom properties for theming
- `assets/icons/` — drop custom SVG icons here
